## Description (4000 chars max)

Gaussian Splatting Viewer brings cutting-edge 3D Gaussian Splatting rendering to your iPhone and iPad — powered by Apple's Metal framework for native, high-performance graphics.

**Native Metal Performance**
Built on the MetalSplatter engine, this app renders Gaussian Splat files directly on-device using Apple's GPU. No cloud rendering, no streaming — just pure local performance.

**Supported Formats**
• .ply — Standard Gaussian Splatting format
• .splat — Compact splat format
• .spz — Compressed splat format (smaller files, faster loads)

**Open Files from Anywhere**
Open .ply, .splat, or .spz files from any iOS Files provider — iCloud Drive, local storage, AirDrop, or third-party apps. The app appears in the system "Open In" menu for all supported file types.

**Key Features**
• Native Metal renderer — smooth 60fps rendering on supported devices
• Open files from any iOS Files provider (iCloud, local, AirDrop, third-party)
• Auto-rotate mode — let your splats spin automatically
• Gesture controls — 1-finger rotate, 2-finger pan, pinch to zoom
• Dark mode native — designed for OLED displays
• No account required — just open and render

**Privacy First**
• No analytics, no tracking, no data collection
• All rendering happens on-device
• Files opened directly from your storage — you control your data

**Requirements**
• iOS 18.0 or later
• iPhone or iPad with A12 chip or later (recommended for smooth rendering)
